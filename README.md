#### dog_feel_classify
画像と音（スペクトログラム）で分類、マルチモーダルTransformer.  
犬の画像(動画) と、鳴き声を同時に入力して、犬の気持ちが分類できる、モデル  

##### 開発環境  

  Ubuntu 24.04 PC  
  PyTorch  

##### 実行環境  

  Ubuntu 24.04 PC or Orange pi 5  
  onnx-runtime  

##### 1. 転移学習 or ファインチューニング  

  dog_feel_train.ipynb  

##### 2. 実行  

  Ubuntu 24.04 PC or Orange pi 5  
  onnx-runtime  
  $ python dog_feel_orangepi_onnx.py
